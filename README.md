# Weather Dashboard

A responsive Weather Dashboard built using HTML, CSS, and JavaScript.

## Features

- Search weather by city
- Displays temperature
- Shows humidity
- Shows wind speed
- Weather condition icons
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API
